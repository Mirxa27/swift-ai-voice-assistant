# Standard Operating Procedure: AI Content Management

This SOP outlines how the team creates and deploys new conversational journeys.

1. **Define the Objective** – Identify the user need or opportunity.
2. **Draft the Prompt** – Write the multi‑stage prompt in the admin panel.
3. **Internal Review** – Share the draft for tone and brand checks.
4. **Sandbox Testing** – Interact with the prompt in the built‑in sandbox.
5. **Staged Rollout** – Deploy to a small group of users for feedback.
6. **Analyze Performance** – Review analytics and user responses.
7. **Full Deployment or Reversion** – Roll out fully if successful or iterate again.

All prompts should follow the Newomen voice guidelines and respect cultural sensitivities.
