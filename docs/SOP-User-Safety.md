# Standard Operating Procedure: User Safety & Crisis Protocol

This protocol defines how Newomen handles potential crisis situations detected by the Guardian agent.

1. **Automated Intervention** – The Guardian agent blocks the reply and displays crisis resources if self‑harm or abuse is detected.
2. **Human Triage** – The on‑call admin reviews the anonymized transcript within 15 minutes.
3. **Confirm and Classify** – Log the incident in the secure report file.
4. **Maintain Privacy** – Do not contact the user directly. Provide only the crisis hotline message.
5. **Post‑Incident Review** – Within 24 hours, check if detection keywords or resources need updates.

This SOP ensures user safety while respecting privacy and legal boundaries.
